# @piar/coming-soon

Componente simple y elegante para mostrar "Coming Soon" en las páginas en desarrollo.

## Uso

```tsx
import { ComingSoon } from '@piar/coming-soon';

export default function Page() {
  return <ComingSoon />;
}
```

## Características

- Diseño minimalista y elegante
- Soporte multiidioma usando @piar/messages
- Sin dependencias pesadas
- Sin tests (paquete simple)
